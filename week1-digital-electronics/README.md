# Week 1 — Digital Electronics Foundations

> Everything in computing — from your phone to a supercomputer to an FPGA — is built on just two values: 0 and 1. This week, you learn why that is, and what you can build from it.

---

## What You Will Learn This Week

By the end of Week 1, you will be able to:

- Convert numbers between binary, decimal, and hexadecimal
- Understand how transistors form logic gates
- Read and construct truth tables
- Simplify Boolean expressions using Karnaugh maps
- Build combinational circuits: adders, multiplexers, encoders, decoders
