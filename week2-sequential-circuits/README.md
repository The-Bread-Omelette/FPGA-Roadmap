# Week 2 — Sequential Circuits & Finite State Machines

> Combinational circuits are stateless. Sequential circuits remember. Add memory to logic, and you get computers.

---

## What Changes This Week

In Week 1, every output was purely a function of current inputs. This week, we introduce **time** and **state**. A sequential circuit's output depends on:

1. The current inputs
2. The current stored state (memory)

This is the foundation of registers, CPUs, communication protocols, and control systems.
